# 🌱 EcoGru

Aplicativo mobile desenvolvido com React Native com foco em sustentabilidade, conscientização ambiental e incentivo a práticas ecológicas no dia a dia.

O projeto busca utilizar a tecnologia como ferramenta para aproximar usuários de hábitos mais sustentáveis, promovendo uma experiência acessível, intuitiva e educativa através de uma aplicação mobile moderna.

---

## 🏅 Premiações

Vale ressaltar que esse foi o projeto ganhador da MostraTech 2024, que é uma feira de ciências desenvolvida com todos os alunos do ensino médio do Colégio ENIAC.

---

## 🚀 Objetivo

O EcoGru foi criado com a proposta de incentivar ações sustentáveis por meio da tecnologia, ajudando usuários a desenvolver maior consciência ambiental e participação em práticas ecológicas.

O projeto busca resolver problemas como:

- falta de incentivo para hábitos sustentáveis;
- dificuldade de acesso a informações ambientais;
- baixa conscientização ecológica no cotidiano;
- ausência de plataformas educativas acessíveis sobre sustentabilidade.

A proposta é transformar a experiência do usuário em algo simples, interativo e conectado com questões ambientais atuais.

---

## 🛠️ Tecnologias Utilizadas

- React Native
- JavaScript
- Git
- GitHub

---

## ✨ Funcionalidades

- Interface mobile intuitiva
- Navegação simples
- Experiência voltada para conscientização ambiental
- Estrutura otimizada para dispositivos móveis
- Organização de conteúdos e informações ecológicas
- Ligação direta via ReactNativeMaps com o Google Maps

---

## 🎯 Visão do Projeto

O EcoGru representa a união entre tecnologia e sustentabilidade, demonstrando como aplicações mobile podem contribuir para conscientização ambiental e incentivo de práticas ecológicas de maneira acessível e moderna.

Além do aspecto social, o projeto também foi desenvolvido como forma de aprimorar conhecimentos em desenvolvimento mobile utilizando React Native.
